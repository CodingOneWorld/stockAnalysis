# 自选股分析模块（selected_stock_analysis）

## 一、整体架构

```
自选股池（CSV / TXT）
        ↓
  股票分类（stock_classification.py）
        ├── 上升通道（超短/短/中/中长/长线）   按斜率 k > 0.2 筛选
        └── 反弹股（开始/中期/晚期）           中长期 k<0 但短期 k>0.1
        ↓
  买点检测（buy_point_detect.py）
        ├── 上升通道回调买点 → 近32日在均线上 + 当前接近 5/10/20/30 均线
        └── 下降通道关键点位 → 接近前低 / 接近 60/140/250 均线
        ↓
  输出到 Word 文档 + CSV（代码、名称、信号描述、日期）
```

---

## 二、文件职责

| 文件 | 功能 | 输入 | 输出 |
|---|---|---|---|
| `main.py` | 主入口，遍历股票池，调用分类+输出 | CSV/TXT 股票池 | 多份 Word 文档 |
| `stock_classification.py` | 按价格斜率把自选股分为上升通道/反弹类 | CSV 股票池 | CSV 分类结果 |
| `rebound_classification.py` | 10日短线反弹检测 | CSV 股票池 | Word 文档 |
| `up_classification.py` | 均线上升通道检测（10/20/50/100日） | 股票代码 | 0/1 信号 |
| `buy_point_detect.py` | 买点信号检测（上升回调 / 下降到关键位） | CSV 股票池 | Word + CSV |
| `key_price_compare.py` | 当前价与均线/极低值比较 | 股票代码 + 均线列表 | 信号列表 |
| `buy_point_effect_tracking.py` | 买点信号回测统计（胜率/平均收益/收益分布） | 信号 CSV | 回测明细 CSV + 统计汇总 CSV |

## 三、核心算法

### 1. 趋势判断（`stock_classification.py`）

用最近 N 天收盘价做**线性回归斜率**（`cal_trend_common` / `cal_stock_price_trend`），以斜率正负和大小区分趋势：

斜率已归一化（以首日价格为基准的累计收益率序列做线性回归），单位：收益率/天，可跨股票比较。

| 类别 | 判断条件（归一化斜率） |
|---|---|
| 超短线上升通道 | k3 > 0.005 且 k5 > 0.005 |
| 短线上升通道 | k5 > 0.005 且 k10 > 0.005 |
| 中线上升通道 | k10 > 0.005 且 k30 > 0.005 |
| 中长线上升通道 | k60 > 0.005 且 k120 > 0.005 |
| 长线上升通道 | k3000 > 0.002 |
| 开始反弹 | k30 < -0.002 且 k5 > 0.002 |
| 中期反弹 | k60 < -0.002 且 k10 > 0.002 |
| 晚期反弹 | k90 < -0.002 且 k30 > 0.002 |

### 2. 均线上升通道检测（`up_classification.py`）

`compare2mean_window(mav, df, latest_days)`：
- 计算收盘价与 mav 日均线的偏差
- 若最近 `latest_days` 天中，低于均线的天数 ≤ 10%，则认为在均线上方（上升通道）

| 通道 | 调用参数 |
|---|---|
| 10日短线 | `is_up_stock(code, mav=12, latest_days=10)` |
| 20日短线 | `is_up_stock(code, mav=22, latest_days=20)` |
| 50日长线 | `is_up_stock(code, mav=30, latest_days=50)` |
| 100日长线 | `is_up_stock(code, mav=30, latest_days=100)` |

### 3. 10日短线反弹检测（`rebound_classification.py`）

同时满足以下条件：
1. k5 > 0，k10 > 0（近10天上涨）
2. k20 < -0.05（近20天下跌，确保是反弹前有下降通道）
3. k100 < 0（中期趋势向下）
4. 最近10天内均在5日均线上方（`compare2mean_window(5, df, 10)`）

### 4. 买点信号检测（`buy_point_detect.py`）

**上升通道回调买点**：
1. 近32日股价均在均线上（`compare2mean_window(32, df, 10) > 0`）
2. 当前价格接近 5/10/20/30 日均线（偏差 ≤ 1%）

**下降通道关键点位**：
1. k5 < 0 且 k20 < 0（处于下降通道）
2. 当前低价接近历史极低点（偏差 ≤ 1%）
3. 或接近 60/140/250 日均线（偏差 ≤ 1%）

### 5. 均线/极低值比较（`key_price_compare.py`）

- `mav_compare_df(df, mav)`：计算当前价与 mav 日均线之差，偏差 ≤ 1% 则触发信号
- `price_compare(cur_price, tag)`：判断价格是否在目标价 ±1% 范围内
- `extreme_price_compare(code)`：计算近300天历史最低价的极值序列，与当前价比较
